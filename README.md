# 🌟 Devesh Sharma's Portfolio

A modern, responsive personal portfolio website built with **HTML**, **CSS**, **Bootstrap**, and **JavaScript**. Featuring a sleek dark theme with glassmorphism design, smooth animations, and interactive elements.

## 🎨 Features

- **Modern Dark Theme** – Sleek glassmorphism design with gradient accents (purple & cyan)
- **Fully Responsive** – Optimized for mobile, tablet, and desktop devices
- **Smooth Animations** – Scroll-reveal effects and hover transitions
- **Interactive Elements** – Email copy functionality, smooth navigation, dynamic modals
- **Professional Sections**:
  - Hero section with quick stats
  - About me with expandable details
  - Skills showcase with animated progress bars
  - Projects gallery with modal details
  - Contact form and social links
  - Floating action buttons (email & call)
  - Back-to-top button

## 🛠️ Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Custom animations, gradients, and glassmorphism effects
- **JavaScript** – Vanilla JS for interactivity
- **Bootstrap 5.3.2** – Responsive grid and components
- **Bootstrap Icons** – Icon library
- **Google Fonts** – Inter font family
